# Cupcake Docs Docs!

Documentation is built with [Zensical](https://zensical.org/)

## Local Development

1. Install [uv](https://docs.astral.sh/uv/) if you haven't got it already
2. Run `uv run zensical serve` to start the docs development server
3. Navigate to [localhost:8000](https://localhost:8000) and get started writing docs!
